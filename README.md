# Network Intrusion Detection System (NIDS)

This project is a Network Intrusion Detection System built using Python.

## Features
- Packet sniffing using Scapy
- Suspicious IP detection
- Live network traffic monitoring
- Interactive bar and pie charts using Plotly

## Technologies Used
- Python
- Scapy
- Pandas
- Plotly
- Jupyter Notebook

## Project Output
The system captures network packets and displays:
- IP traffic bar chart
- Traffic distribution pie chart
- Suspicious IP list
